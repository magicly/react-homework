#### 运行方法

1. 通过git clone命令将项目下载到本地文件夹
> git clone 项目Git地址
2. 运行CMD命令，cd进入项目的本地文件夹中
> cd 项目所在文件夹
3. cd到6-redux\wsyun文件夹
> `cd 6-redux\wsyun`
4. 执行命令 npm install 
> `npm install`
5. 执行命令 npm start
> `npm start`
